# SiriusIQ-Demo
*Demo built for [SiriusIQ Mobile](http://www.siriusiqmobile.com)*

## Installation ##

Requirements: 
- node.js (nodemon helps) 
- npm package manager
- bower package manager
- git

```
Terminal: 

$ npm install
$ bower install

// to start

node server.js  // nodemon server.js also fine
```
